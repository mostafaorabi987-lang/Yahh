# متابع اللوحات

تطبيق ويب (يعمل زي تطبيق موبايل، وممكن تثبّته على شاشة موبايلك) لمتابعة سياراتك بأرقام اللوحات، مع بحث صوتي بالعربي، بحث بالكاميرا (OCR)، وتحديد موقع كل سيارة على الخريطة — ويشتغل **حتى بدون إنترنت**.

## المميزات
- إضافة أكتر من ١٢ سيارة (رقم اللوحة حقل مطلوب).
- بحث صوتي بالعربي: تتكلم وهو يكتب الكلام في مربع البحث لحظة بلحظة، ويقولك بصوت عربي "موجودة" أو "مش موجودة". (يحتاج إنترنت — تفصيل تحت).
- **بحث بالكاميرا (OCR)**: تصوّر لوحة سيارة بكاميرا الموبايل، والتطبيق يقرأ الحروف والأرقام العربية منها تلقائيًا، ويدوّر عليها في قائمة سياراتك. يشتغل بدون إنترنت بعد أول استخدام.
- تحديد موقع كل سيارة على خريطة حقيقية (اضغط، اسحب الدبوس، أو استخدم موقعك الحالي).
- **يعمل بدون إنترنت (Offline Mode)**: تقدر تمسح لوحات وتحدد مواقع في أماكن الإشارة الضعيفة، والبيانات بتتحفظ على جهازك فورًا. لما الإنترنت يرجع، التطبيق بيكمّل تلقائيًا العناوين المقروءة لأي موقع اتحفظ بإحداثيات بس.
- واجهة عربية بالكامل (RTL) بتصميم يشبه اللوحات الرسمية.

## ملفات المشروع
دلوقتي المشروع بقى فيه أكتر من ملف واحد (عشان يقدر يشتغل أوفلاين ويتثبّت كتطبيق):
- `index.html` — التطبيق نفسه.
- `manifest.json` — بيانات تثبيت التطبيق (اسم، أيقونة، ألوان).
- `sw.js` — الـ Service Worker المسؤول عن تخزين الملفات وتشغيل التطبيق أوفلاين.
- `icon.svg` — أيقونة التطبيق.

**كل الملفات الأربعة لازم تترفع مع بعض في جذر نفس الـ Repository.**

## طريقة النشر على GitHub Pages

1. اعمل Repository جديد على GitHub (public).
2. ارفع الملفات الأربعة (`index.html`, `manifest.json`, `sw.js`, `icon.svg`) لجذر الـ Repository (يعني في المسار الرئيسي مش جوه مجلد فرعي).
3. من إعدادات الريبو: **Settings → Pages**.
4. تحت **Build and deployment** اختار **Deploy from a branch**.
5. اختار البرانش `main` والمجلد `/ (root)`، ثم اضغط **Save**.
6. بعد دقيقة أو اتنين هيديك رابط شكله:
   `https://USERNAME.github.io/REPO-NAME/`

افتح الرابط من الموبايل أو الكمبيوتر وابدأ تستخدم التطبيق. **مهم:** أول مرة لازم تفتحه وانت متصل بالإنترنت عشان يحمّل ويخزّن كل حاجة محتاجها (الخطوط، الخريطة، قارئ اللغة العربية)؛ بعد كده هيشتغل أوفلاين.

### لتثبيت التطبيق على الموبايل (اختياري بس بينصح بيه)
- **أندرويد (Chrome)**: افتح الرابط، هيظهرلك اقتراح "Add to Home screen" أو من قائمة المتصفح ⋮ اختار "Install app".
- **آيفون (Safari)**: افتح الرابط، دوس زر المشاركة، واختار "Add to Home Screen".
بعد التثبيت هيفتح بدون شريط عنوان زي أي تطبيق عادي، وهيشتغل أوفلاين بشكل أفضل.

## ملاحظات مهمة

- **البحث الصوتي** يحتاج HTTPS (GitHub Pages بيوفره تلقائي)، إذن استخدام الميكروفون، **واتصال إنترنت فعّال** — لأن محرك التعرف على الصوت في المتصفح (خصوصًا Chrome) بيبعت الصوت لسيرفرات جوجل يحلله، فمش هيشتغل أوفلاين. لو مفيش نت، التطبيق هيقولك تستخدم الكاميرا أو الكتابة اليدوية بدلًا منه.

- **البحث بالكاميرا (OCR)** يستخدم مكتبة Tesseract.js اللي بتشتغل بالكامل جوه المتصفح (مفيش صور بترفع لأي سيرفر). أول مرة تستخدمها لازم يكون فيه نت عشان يحمّل بيانات قراءة اللغة العربية (كام ميجا)؛ بعد المرة دي بيتخزنوا على الجهاز ويشتغلوا حتى بدون إنترنت. النتيجة بتتحط في مربع البحث اليدوي تلقائيًا فتقدر تعدّلها لو القراءة مش مظبوطة ١٠٠٪.

- **البيانات (السيارات ومواقعها) بقت بتتحفظ بشكل دائم على جهازك** (باستخدام تخزين المتصفح المحلي)، مش بس في الجلسة زي الأول. تفضل موجودة حتى لو قفلت المتصفح أو الموبايل، وحتى وانت أوفلاين تمامًا.

- **مهم جدًا عن "المزامنة" — التطبيق مفيهوش سيرفر أو قاعدة بيانات مركزية.** يعني:
  - البيانات محفوظة على **الجهاز اللي فتحت بيه التطبيق فقط**، ومش بتتشارك تلقائيًا بين موبايلك وموبايل حد تاني.
  - "المزامنة" اللي بتحصل لما النت يرجع، معناها إن أي موقع اتحفظ أوفلاين بس بإحداثيات (من غير كتابة عنوان)، التطبيق بيدوّر له تلقائيًا على عنوان مقروء بالعربي (زي "شارع الجمهورية، بني سويف") لحظة رجوع الاتصال، وهيظهرلك شارة "⏳ قيد المزامنة" لحد ما ده يحصل.
  - لو محتاج إن أكتر من شخص/جهاز يشوفوا نفس القائمة ويحدّثوها مع بعض (مزامنة حقيقية بين أجهزة)، ده محتاج إضافة سيرفر/قاعدة بيانات خلفية — مش موجود حاليًا، وممكن نضيفه لو حبيت.

- عند فتح شاشة تحديد الموقع وانت أوفلاين، خريطة الشوارع (البلاطات) ممكن ما تظهرش كاملة إلا لو كانت المنطقة دي اتفتحت قبل كده وهي متخزنة، لكن تقدر تستخدم "موقعي الحالي" (GPS بيشتغل بدون نت) أو تكتب عنوان يدويًا وهيتحفظ عادي.
